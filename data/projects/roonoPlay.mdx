# Roono Play

<div className="project-overview">

## 심리테스트 플랫폼 & 데이터 분석 서비스

Roono Play는 단독 개발한 웹 애플리케이션으로, 다양한 심리테스트 제공과 사용자 응답 데이터 수집·분석 기능을 갖췄습니다. 주요 기능은 테스트별 고유 디자인, 응답 데이터 분석, 관리자용 테스트 생성·관리, 결과 캐싱 및 SNS 공유입니다. Vercel 배포, Cloudinary 이미지 저장에 사용했습니다.

</div>

<div className="tech-stack">

### 사용 기술

- **Frontend**: Next.js 14(App Router), TypeScript, SCSS + CSS Modules, Zustand, Axios, Recharts, React Icons
- **Backend**: Node.js, Next.js API Routes, PostgreSQL(Supabase), Prisma, NextAuth.js, bcryptjs
- **DevOps & 배포**: Vercel, Cloudinary, Vercel Analytics, Nodemailer

</div>

## 핵심 문제상황

<div className="problem-solution">

<div className="challenge-item">

### 테스트별 고유 디자인과 확장성 문제

- **문제**: 테스트별 디자인과 로직을 유지하면서, 신규 테스트를 추가할 수 있는 구조가 필요. 기존 방식은 테스트마다 UI·로직이 분리되어 있어 관리가 복잡했고, 유지보수 시 동일한 작업을 반복해야했음
- **기술 해결**: 메타데이터(제목·설명·이미지·색상 등) 기반 구조로 로직을 고정하고 관리자 페이지에서 데이터만 수정 가능하도록 설계. SCSS 테마 시스템으로 다양한 브랜딩을 지원.
- **결과**: 테스트별 브랜드 경험 제공, 구조 재사용성 확보, 유지보수성 향상.

</div>

<div className="challenge-item">

### 관리자 기능의 직관성 및 보안

- **문제**: 관리자 인터페이스가 복잡하여 생산성이 저하, 인증·데이터 보안이 취약하여 데이터 유출 위험이 생김
- **기술 해결**: NextAuth.js와 bcryptjs를 활용해 관리자 인증과 세션 기반 권한 관리를 구현. 3단계 테스트 생성 프로세스를 도입해 직관적인 UI/UX를 제공. 또한 API 엔드포인트에 권한 검증 로직을 적용하고 IP와 User Agent를 수집해 사용자 추적과 데이터 무결성을 확보
- **결과**: 관리자 사용 편의성 향상, 보안 위협 감소, 작업 효율성 60% 증가. 생산성 100% 증가.

</div>
