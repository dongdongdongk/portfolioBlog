# MoHotel

<div className="project-overview">

## 호텔 관리 PMS & 호텔 예약 웹 서비스

MoHotel은 팀 프로젝트로 개발한 호텔 통합 관리 시스템으로 주요 기능은 실시간 객실 현황 관리, Spring Scheduler 기반 자동화 체크아웃/청소 요청 시스템, FullCalendar API를 활용한 청소팀 일정 관리, 카카오 상담 API 실시간 고객지원, Google SMTP 기반 ID/PW 찾기, Apache POI Excel 다운로드, WebSocket 실시간 알림 시스템입니다. Ajax로 동적 UI 업데이트, Google Chart API로 매출 통계 시각화, UUID 기반 임시 비밀번호 생성으로 보안 강화를 구현했습니다.



</div>

<div className="challenge-item">

### 동시다발적 객실 상태 변경으로 인한 데이터 정합성 오류

- **문제**: 체크아웃, 청소 요청, 예약 등이 동시에 발생하면서 같은 객실에 대해 상태가 충돌하는 문제 발생. 특히 일괄 체크아웃 스케줄러 실행 중 수동 상태 변경 시 데이터 불일치로 이중 예약 발생.
- **기술 해결**: 트랜잭션 격리 수준을 READ_COMMITTED로 설정하고, 객실 상태 변경 시 비관적 락(Pessimistic Lock) 적용. Spring Scheduler와 수동 변경 간의 동기화를 위한 분산 락 메커니즘 구현
- **결과**: 데이터 정합성 오류 100% 해결, 이중 예약 사고 제로 달성, 객실 운영 안정성 확보.
</div>

<div className="challenge-item">

### Ajax 비동기 통신 시 세션 만료 처리 미비

- **문제**: 실시간 청소 요청 및 객실 상태 업데이트 중 사용자 세션이 만료되면 Ajax 요청이 실패하지만 안내 없이 화면이 동작하지 않아 사용자 혼란 발생.
- **기술 해결**: Ajax 글로벌 에러 핸들러 설정으로 401 Unauthorized 응답 시 자동 로그인 페이지 리다이렉트 구현. 세션 만료 5분 전 경고 알림 모달 추가, 세션 연장 버튼으로 사용자 편의성 개선.
- **결과**: 세션 만료로 인한 사용자 혼란 해결, 명확한 상태 안내로 사용자 경험 개선.

</div>
